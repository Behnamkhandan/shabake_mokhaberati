# مدولاسیون SSB (Single Sideband)
## یک توضیح جامع و کامل برای درس شبکه‌های مخابراتی

## 1. مقدمه و تاریخچه

### 1.1. تعریف
**مدولاسیون SSB (Single Sideband)** یا **تک‌باند کناری** یک تکنیک مدولاسیون دامنه پیشرفته است که در آن تنها یکی از دو باند کناری (بالایی یا پایینی) همراه با حامل سرکوب‌شده ارسال می‌شود.

### 1.2. تاریخچه
- **۱۹۱۵**: اولین بار توسط جان رنسام کارسون (John Renshaw Carson) ابداع شد
- **۱۹۲۳**: ثبت اختراع توسط کارسون
- **دهه ۱۹۳۰**: اولین استفاده عملی در مخابرات دریایی
- **دهه ۱۹۵۰**: استفاده گسترده در ارتباطات رادیویی HF

## 2. مبانی نظری

### 2.1. مدولاسیون دامنه معمولی (AM)
در مدولاسیون AM استاندارد:
- سیگنال مدوله‌شده شامل سه جزء است:
  1. حامل اصلی (Carrier)
  2. باند کناری بالایی (USB - Upper Sideband)
  3. باند کناری پایینی (LSB - Lower Sideband)

### 2.2. مشکل AM استاندارد
- **کارایی توان پایین**: ۶۶٪ توان در حامل تلف می‌شود
- **پهنای باند مضاعف**: هر دو باند کناری اطلاعات یکسانی دارند
- **حساسیت به نویز**: پهنای باند زیاد باعث دریافت نویز بیشتر می‌شود

### 2.3. راه‌حل SSB
در SSB:
- حامل کاملاً سرکوب می‌شود (DSB-SC)
- تنها **یکی** از باندهای کناری ارسال می‌شود
- در نتیجه:
  - صرفه‌جویی ۵۰٪ در پهنای باند
  - صرفه‌جویی در توان ارسالی
  - کاهش تداخل بین کانال‌ها

## 3. تحلیل ریاضی

### 3.1. سیگنال پیام
فرض کنید سیگنال پیام ما:
```
m(t) = A_m cos(ω_m t)
```

### 3.2. سیگنال حامل
سیگنال حامل:
```
c(t) = A_c cos(ω_c t)
```

### 3.3. سیگنال DSB-SC
سیگنال مدوله‌شده DSB-SC:
```
s_DSB(t) = m(t) * c(t) = A_m A_c cos(ω_m t) cos(ω_c t)
```

با استفاده از اتحاد مثلثاتی:
```
s_DSB(t) = (A_m A_c)/2 [cos((ω_c + ω_m)t) + cos((ω_c - ω_m)t)]
```

### 3.4. سیگنال SSB
برای تولید SSB، نیاز به تبدیل هیلبرت داریم:

**تبدیل هیلبرت**:
```
H[cos(ωt)] = sin(ωt)
H[sin(ωt)] = -cos(ωt)
```

سیگنال SSB-USB:
```
s_USB(t) = m(t)cos(ω_c t) - m̂(t)sin(ω_c t)
```

سیگنال SSB-LSB:
```
s_LSB(t) = m(t)cos(ω_c t) + m̂(t)sin(ω_c t)
```

که در آن `m̂(t)` تبدیل هیلبرت `m(t)` است.

### 3.5. نمایش طیفی
```
DSB-SC:     [---LSB---] C [---USB---]
SSB-USB:                  C [---USB---]
SSB-LSB:    [---LSB---] C
```

## 4. روش‌های تولید SSB

### 4.1. روش فیلترینگ (متداول‌ترین روش)
1. ابتدا سیگنال DSB-SC تولید می‌شود
2. سپس با فیلتر مناسب یکی از باندها حذف می‌شود
3. **مزایا**: ساده‌سازی مدار
4. **معایب**: نیاز به فیلترهای بسیار تیز

### 4.2. روش فازینگ (Weaver Method)
1. استفاده از دو مدولاتور ۹۰ درجه اختلاف فاز
2. حذف یکی از باندها با جمع/تفریق مناسب
3. **مزایا**: نیاز به فیلترهای ساده‌تر
4. **معایب**: پیچیدگی مدار بیشتر

### 4.3. روش تبدیل هیلبرت
1. تولید مستقیم سیگنال آنالیتیک
2. استفاده از فرمول‌های ریاضی SSB
3. **مزایا**: دقیق‌ترین روش
4. **معایب**: نیاز به پردازش سیگنال دیجیتال

## 5. دمدولاسیون SSB

### 5.1. شرط موفقیت‌آمیز بودن دمدولاسیون
برای دمدولاسیون صحیح SSB نیاز است:
1. **همان‌سازی فرکانس** (Frequency Synchronization)
2. **همان‌سازی فاز** (Phase Synchronization)
3. **پایداری اسیلاتور محلی**

### 5.2. روش‌های دمدولاسیون

#### الف) روش حامل بازیابی‌شده (Pilot Carrier)
- یک سیگنال حامل کوچک (پایلوت) همراه SSB ارسال می‌شود
- گیرنده از این حامل برای سنکرون‌سازی استفاده می‌کند
- **کاربرد**: ارتباطات HF و رادیو آماتور

#### ب) روش Costas Loop
- استفاده از حلقه قفل فاز (PLL)
- بازیابی خودکار فرکانس و فاز
- **کاربرد**: سیستم‌های دیجیتال

#### ج) روش مربع‌کننده
- مربع کردن سیگنال ورودی
- استخراج فرکانس حامل از مؤلفه‌ی ۲f_c
- **کاربرد**: سیستم‌های ساده

### 5.3. فرمول دمدولاسیون
اگر سیگنال دریافتی SSB باشد:
```
r(t) = m(t)cos(ω_c t) ∓ m̂(t)sin(ω_c t)
```

و اسیلاتور محلی:
```
c_local(t) = cos(ω_c t + φ)
```

خروجی ضرب‌کننده:
```
v(t) = r(t) * c_local(t)
```

بعد از فیلتر پایین‌گذر:
```
y(t) = (1/2)m(t)cos(φ) ± (1/2)m̂(t)sin(φ)
```

اگر φ = ۰ (همان‌سازی کامل فاز):
```
y(t) = (1/2)m(t)  ✅
```

## 6. محاسبات و مقایسه‌ها

### 6.1. محاسبه پهنای باند
```
AM استاندارد:    B_AM = 2 * f_m
DSB-SC:          B_DSB = 2 * f_m
SSB:             B_SSB = f_m
```

که در آن `f_m` بیشترین فرکانس سیگنال پیام است.

### 6.2. محاسبه توان
**AM استاندارد**:
```
P_AM = P_c + P_USB + P_LSB
     = P_c + (1/2)m_a²P_c + (1/2)m_a²P_c
     = P_c(1 + m_a²)
```

**SSB**:
```
P_SSB = P_Sideband = (1/4)m_a²P_c
```

**راندمان**:
```
η_AM = (m_a²)/(2 + m_a²) × 100%
برای m_a = 1: η_AM ≈ 33%

η_SSB = 100% (تمام توان در اطلاعات)
```

### 6.3. محاسبه SNR
نسبت سیگنال به نویز در خروجی:

**AM**:
```
(SNR)_out_AM = (m_a²)/(1 + m_a²) × (SNR)_in
```

**SSB**:
```
(SNR)_out_SSB = (SNR)_in
```

## 7. مزایا و معایب SSB

### 7.1. مزایا
1. **صرفه‌جویی در پهنای باند** (۵۰٪)
2. **کارایی توان بالا** (تمام توان برای اطلاعات)
3. **کاهش تداخل** بین کانال‌های مجاور
4. **کاهش نویز** دریافتی (پهنای باند کمتر)
5. **قابلیت Fading کمتر** نسبت به AM
6. **امنیت نسبی** (دریافت نیاز به سنکرون‌سازی دارد)

### 7.2. معایب
1. **پیچیدگی مدار** مدولاتور و دمودولاتور
2. **نیاز به سنکرون‌سازی دقیق** فرکانس و فاز
3. **حساسیت به اعوجاج فاز**
4. **مشکل در انتقال سیگنال‌های DC**
5. **هزینه بیشتر** نسبت به AM معمولی

### 9.1. استانداردهای ITU
- **ITU-R M.489-2**: مشخصات فنی SSB برای مخابرات دریایی
- **ITU-R SM.1138**: استفاده از SSB در باند HF

### 9.2. مشخصات فنی معمول
1. **پهنای باند صوتی**: ۳۰۰-۳۰۰۰ هرتز
2. **انحراف فرکانس مجاز**: ±۵۰ هرتز
3. **سرکوب حامل**: معمولاً ۴۰-۶۰ دسی‌بل
4. **سرکوب باند کناری نامطلوب**: ۵۰-۷۰ دسی‌بل

### 9.3. انواع SSB
1. **SSB-SC**: Single Sideband Suppressed Carrier
2. **SSB-RC**: Single Sideband Reduced Carrier
3. **SSB-FC**: Single Sideband Full Carrier (نادر)

## 10. پیاده‌سازی عملی

### 10.1. مدارهای آنالوگ
1. **مدولاتور متوازن** (Balanced Modulator)
2. **فیلتر کوارتز** یا **مکانیکی**
3. **مدار فازینگ** با اختلاف ۹۰ درجه
4. **تقویت‌کننده‌های خطی**

### 10.2. پیاده‌سازی دیجیتال (DSP)
```python
# شبه کد تولید SSB با DSP
def generate_ssb(signal, fc, fs, mode='USB'):
    # تبدیل هیلبرت دیجیتال
    hilbert_signal = hilbert_transform(signal)
    
    # تولید حامل
    t = np.arange(len(signal)) / fs
    carrier_i = np.cos(2 * np.pi * fc * t)
    carrier_q = np.sin(2 * np.pi * fc * t)
    
    if mode == 'USB':
        ssb = signal * carrier_i - hilbert_signal * carrier_q
    else:  # LSB
        ssb = signal * carrier_i + hilbert_signal * carrier_q
    
    return ssb
```

### 10.3. آی‌سی‌های اختصاصی
1. **MC1496**: مدولاتور متوازن
2. **NE602**: میکسر و اسیلاتور
3. **DSP پردازنده‌ها**: برای پیاده‌سازی دیجیتال

## 11. مشکلات و چالش‌ها

### 11.1. مشکل سنکرون‌سازی
- **اثر انحراف فرکانس**: اگر Δf انحراف باشد:
  ```
  خروجی: m(t)cos(2πΔf t) ∓ m̂(t)sin(2πΔf t)
  ```
  - باعث اعوجاج و تغییر تن صدا می‌شود

### 11.2. مشکل اعوجاج فاز
- تغییر فاز غیرخطی در کانال
- ایجاد اعوجاج در سیگنال بازیابی‌شده

### 11.3. مشکل سیگنال‌های کم‌فرکانس
- فیلترهای SSB معمولاً فرکانس‌های زیر ۳۰۰ هرتز را حذف می‌کنند
- مشکل در انتقال سیگنال‌های DC

## 13. توسعه‌های آینده و کاربردهای نوین

### 13.1. SSB در ارتباطات دیجیتال
- **OFDM**: استفاده از SSB در زیرحامل‌ها
- **Cognitive Radio**: انتخاب خودکار بین USB/LSB
- **Software Defined Radio (SDR)**: پیاده‌سازی نرم‌افزاری

### 13.2. SSB در فیبر نوری
- **SSB-OFDM**: برای کاهش dispersion
- **مدولاسیون coherent**: استفاده از SSB در مدولاسیون فاز

### 13.3. کاربرد در پردازش سیگنال
- **کاهش aliasing** در نمونه‌برداری
- **فشرده‌سازی طیف** در سیستم‌های چندحاملی
